# GOAL-01 Coding Prompt

Implement only the repository foundation.

Allowed files:

- `AGENTS.md`
- `README.md`
- `BUSINESS.md`
- `SYSTEM.md`
- `SPEC.md`
- `GOALS.md`
- `PLAN.md`
- `docs/**`
- `implementation-goals/**`
- `scripts/**`
- `k8s/**` placeholders
- `reports/validation/**`

Forbidden:

- product storefront source;
- payment/order/stock code;
- production secret values;
- live deployment.

Expected output:

- docs and gates exist;
- validation evidence recorded;
- foundation commit created.
